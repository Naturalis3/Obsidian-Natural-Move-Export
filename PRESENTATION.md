# Natural Move/Export: Video/GIF Showcase Storyboard

To create a great presentation for the Obsidian community, follow this script to record a short (30-60s) video or a series of GIFs.

## Scene 1: The "Natural Move" (Free)
1.  **Action:** Open the File Explorer in Obsidian.
2.  **Action:** Select 2-3 Markdown files.
3.  **Action:** Press `Cmd + C` (Mac) or `Ctrl + C` (Win).
4.  **Action:** Switch to your Desktop or a folder in Finder/Explorer.
5.  **Action:** Press `Cmd + V` (Mac) or `Ctrl + V` (Win).
6.  **Result:** The files appear instantly outside Obsidian.
7.  **Caption:** "Copy files from Obsidian to your OS as if they were in a normal folder."

## Scene 2: The "Alt-Drag" (Free)
1.  **Action:** Hold the `Alt` key.
2.  **Action:** Click and drag a file from Obsidian's File Explorer.
3.  **Action:** Drop it into another app (e.g., Mail, Slack, or a folder).
4.  **Result:** The file is copied instantly.
5.  **Caption:** "Hold Alt to drag & drop files directly out of Obsidian."

## Scene 3: Professional Export (Pro)
1.  **Action:** Right-click a Markdown file.
2.  **Action:** Hover over "Word Templates".
3.  **Action:** Select a template (e.g., "Company_Report").
4.  **Result:** A notification appears: "Exporting...", then "Success!".
5.  **Action:** Open the target folder to show the perfectly formatted Word document.
6.  **Caption:** "Export to Word using your own custom .docx templates."

## Scene 4: Folder Handling (Pro)
1.  **Action:** Select an entire folder in Obsidian.
2.  **Action:** Right-click -> "Copy to target folder".
3.  **Result:** The entire folder structure is mirrored to your backup/export location.
4.  **Caption:** "Manage entire folders and sync them to your target directory."

---

### Tips for a great video:
- Use a clean Obsidian theme (e.g., Default or Minimal).
- Zoom in on the notifications and context menu.
- Use a screen recording tool like **ScreenStudio** (Mac) or **OBS** to capture both Obsidian and the Desktop.
