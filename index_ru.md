**Natural Move/Export** — это плагин для Obsidian, который плавно устраняет разрыв между Obsidian и вашей операционной системой. Он делает так, что проводник файлов Obsidian ощущается как нативная папка на вашем компьютере.

## Почему Natural Move/Export?

Проводник файлов Obsidian мощный, но изолированный. Копирование файлов из Obsidian обычно требует сначала открытия папки в Finder/Проводнике. **Natural Move/Export** исправляет это, обеспечивая нативное общесистемное копирование-вставку, перетаскивание (drag & drop) и профессиональный экспорт непосредственно из Obsidian.

## ✨ Ключевые особенности

### 📋 Нативное системное копирование (Free и Pro)
Выберите файлы в Obsidian и нажмите `Cmd + C` (Mac) или `Ctrl + C` (Win). Затем вы можете вставить их прямо на рабочий стол или в любое другое приложение с помощью `Cmd + V`.
*   **Free:** Копирование отдельных файлов.
*   **Pro:** Копирование целых папок и их содержимого.

### 🖱️ Прямое перетаскивание (Free и Pro)
Удерживая клавишу `Alt`, перетаскивайте файлы или папки из Obsidian прямо в другие приложения (например, Почту, Slack или папку).
*   **Free:** Перетаскивание отдельных файлов.
*   **Pro:** Перетаскивание целых папок.

### 🚀 Профессиональный экспорт через Pandoc (Pro)
Превратите свои заметки Markdown в отполированные документы одним щелчком мыши.
*   **Word (.docx):** Используйте собственные шаблоны `.docx` для профессионального брендинга.
*   **PowerPoint (.pptx):** Мгновенно превращайте заметки в презентации.
*   **PDF и Beamer:** Высококачественный академический экспорт и экспорт слайдов.
*   **HTML и Markdown:** Чистый, автономный экспорт.

### 📁 Синхронизация с целевой папкой (Pro)
Настройте фиксированную целевую папку в настройках. Копируйте туда файлы и папки одним щелчком мыши через контекстное меню. Идеально подходит для резервного копирования, обмена или экспорта проектов.

### 🌍 Умная локализация и определение ОС
*   **Автоматический язык:** Плагин автоматически адаптируется к языковым настройкам вашего Obsidian.
*   **Умное определение ОС:** Настройки и шаблоны путей автоматически подстраиваются в зависимости от того, используете ли вы Windows, macOS или Linux.

### Как активировать Pro
1. Купите лицензионный ключ на [Lemon Squeezy](https://your-store.lemonsqueezy.com).
2. Откройте Obsidian и перейдите в **Настройки > Natural Move**.
3. Введите лицензионный ключ в поле **License Key**.
4. Нажмите **Verify**.
5. После активации все функции Pro будут разблокированы немедленно.

## 🛠️ Предварительные условия для экспорта через Pandoc

Для использования функций экспорта (Word, PowerPoint, PDF и т. д.) в вашей системе должен быть установлен **Pandoc**.

- **Mac:** `brew install pandoc` (через Homebrew) или загрузите установщик с [сайта Pandoc](https://pandoc.org/installing.html).
- **Windows:** Загрузите установщик с сайта Pandoc.

### ⚠️ Важно для экспорта в PDF и Beamer (MacTeX)

Pandoc требует наличия дистрибутива LaTeX в фоновом режиме для создания PDF-файлов и презентаций Beamer. На Mac стандартом является **MacTeX**.

**Установка MacTeX (Mac):**

*Вариант 1: Через Homebrew (рекомендуется)*
Откройте терминал и введите следующую команду:
```bash
brew install --cask mactex-no-gui
```
*(Примечание: загрузка очень большая (около 5 ГБ), так как она содержит все необходимые пакеты LaTeX. Версия `mactex-no-gui` устанавливает только инструменты командной строки, чего достаточно для Pandoc).*

*Вариант 2: Ручная загрузка*
1. Перейдите на официальный сайт: [tug.org/mactex](https://www.tug.org/mactex/)
2. Загрузите файл `MacTeX.pkg` и запустите установщик.

После установки Pandoc и MacTeX вам может потребоваться полностью перезапустить Obsidian, чтобы новые системные пути были распознаны. Плагин Natural Move/Export автоматически ищет необходимые программы в стандартных путях (`/Library/TeX/texbin`, `/opt/homebrew/bin`, `/usr/local/bin`).

## Установка в Obsidian

### Через BRAT (бета-тестирование)
1. Установите плагин "Obsidian 42 - BRAT" из сообщества плагинов.
2. Перейдите в настройки BRAT и нажмите "Add Beta Plugin".
3. Введите URL этого репозитория GitHub.
4. Нажмите "Add Plugin".

### Ручная установка
1. Загрузите проект в виде ZIP-архива и распакуйте его.
2. Создайте папку в вашем хранилище Obsidian: `.obsidian/plugins/natural-move`
3. Скопируйте все извлеченные файлы проекта в эту новую папку.
4. Откройте терминал в этой папке и выполните следующие команды:
   ```bash
   npm install
   npm run build
   ```
5. Откройте Obsidian и перейдите в **Настройки > Сообщество плагинов**.
6. Отключите "Безопасный режим", если вы еще этого не сделали.
7. Нажмите "Обновить" и включите плагин **Natural Move/Export**.
